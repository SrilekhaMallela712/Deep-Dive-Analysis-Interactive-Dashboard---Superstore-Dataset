# Deep-Dive-Analysis-Interactive-Dashboard---Superstore-Datase
📌 Project Overview
This project performs a deep-dive business analysis on the Sample Superstore dataset.
The goal is to identify key performance trends, analyze profitability, and build an interactive dashboard using Tableau.
🎯 Objective
To answer complex business questions by:
Defining Core KPIs
Performing detailed data analysis
Building an interactive dashboard for business insights
📂 Dataset
Dataset: SampleSuperstore.csv
Source: Kaggle
Records include Sales, Profit, Category, Region, Segment, City, etc.
📈 Core KPIs Defined
Total Sales
Formula: SUM(Sales)
Total Profit
Formula: SUM(Profit)
Profit Margin (%)
Formula: (Total Profit / Total Sales) × 100
Total Quantity Sold
Formula: SUM(Quantity)
🔍 Deep-Dive Analysis Performed
1️⃣ Sales by Category
Technology has the highest sales.
Furniture and Office Supplies follow.
2️⃣ Sales by Segment
Consumer segment contributes the highest revenue.
3️⃣ Profit by Region
West region generates the highest profit.
Central region shows lower profitability.
4️⃣ Top 5 Cities by Sales
Identified highest revenue-generating cities.
📊 Dashboard Features (Tableau)
Interactive filters (Category, Region, Segment)
KPI summary cards
Bar charts for category & segment analysis
Profit visualization using color indicators
Top 5 cities view
Clean and professional dashboard layout
